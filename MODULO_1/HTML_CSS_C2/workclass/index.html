Calsificación elementos HTML

Elementos de bloque

Son los elementos que ocupan todo el ancho disponible que tienen y adicional agregan saltos de linea antes y despues del elemento

  • div
  • p
  • h1 ...h6
  • ul
  • ol
  • li

Elementos en linea

Son los elementos que ocupan el ancho del contenido que tienen y no agregan saltos de linea antes ni despues del elemento

  • img algo mas de texto
  • b
  • strong
  • em
  • u
  • span

Elementos semanticos

Son los elementos que describen la estructura del documento y ayudan con la accesibilidad y el SEO

  • header
  • nav
  • footer
  • article
  • secition
  • aside
